# WebDAV-AudioPlayer
A simple AudioPlayer based on [CSCore](https://github.com/filoe/cscore) to play music from a WebDAV location using [WebDav.Client](https://github.com/skazantsev/WebDavClient).

The following codecs are supported:
* MP3
* WAVE
* FLAC
* AAC
* AC3
* WMA
* OGG-Vorbis
